# VVD 3.0 - десктоп-клиент (Windows)

Тонкий десктоп-клиент CRM VVD 3.0 на Electron. По сути это окно, в котором
открывается боевой сайт `https://formulavvd.com` - весь функционал тот же, что
и на сайте. Вход сохраняется между запусками.

При запуске сразу открывается страница входа (если пользователь не авторизован)
или дашборд `/home` (если уже вошёл). Лендинг в приложении не показывается.

## Требования

- Node.js 20+ (рекомендуется LTS)
- Windows - для сборки установщика `.exe`

## Установка зависимостей

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm start
```

Откроется окно приложения с сайтом. Это самый быстрый способ проверить клиент -
сборка установщика для этого не нужна.

## Сборка установщика для Windows

1. Сгенерировать иконку `.ico` из логотипа (один раз или после смены логотипа):

   ```bash
   npm run icon
   ```

2. Собрать установщик:

   ```bash
   npm run dist
   ```

Готовый установщик появится в папке `dist/` - файл вида
`VVD-3.0-Setup-1.0.0.exe`. Его можно запускать и устанавливать.

> Примечание: приложение пока без цифровой подписи, поэтому при первой установке
> Windows может показать предупреждение SmartScreen «неизвестный издатель» -
> нужно нажать «Подробнее → Выполнить всё равно». На работу это не влияет.

## Структура

```
vvd-desktop/
├── src/main.js          # основной процесс: окно, меню, внешние ссылки
├── src/preload.js       # заготовка preload (пока пустая)
├── build/icon.png       # логотип-источник (512×512)
├── build/icon.ico       # иконка Windows (генерируется из icon.png)
├── electron-builder.yml # конфигурация сборки
└── package.json
```
